Apollon
=====================================

Apollon is a PoS-based cryptocurrency based on [PIVX](https://github.com/PIVX-Project/PIVX).

More information at 
[apollon.io](http://www.apollon.io)  
[Discord](https://discordapp.com/invite/t3ZwaSW)
